本书涵盖了Ruby on Rails绝大部分的常用功能以及介绍，为了兼顾入门及实战，本书拆分成两个部分：Part 1前八章为入门实作，逐步实作一个活动管理系统、Part 2深入剖析各种元件和高阶议题。本书期许可以让有些许经验的网页程式设计师，可以在本书中学习Rails之道，将之实际应用在工作之中。

本书体例

为了避免读者混淆指令的使用情境，区分如下：

  • 钱号$开头的是指命令列下的指令
  • 大于>开头的是指在rails console主控台下的指令
  • 井号#是註解说明
  • 其他则是程式原始码

例如在以下的操作中，我们先输入了rails console进入Rails的主控台，然后输入Person.first和Person.count：

那么在本书中，会这样书写：

$ rails console
Loading development environment (Rails 4.2.0)
> Person.first # <Person id: 1>
> Person.count # 1

本书对象

本书主要对象为网站程式设计师，比较不适合完全没有网页开发经验的朋友，在学习Rails前，最好对以下知识有基本认识：

  • 一些程式设计的经验，例如Java、C#、PHP、Python、Perl、JavaScript
  • HTML/CSS网页设计基础，推荐Codecademy或Code School教材
  • 对网络通讯协定有基础认识，例如HTTP、DNS等，推荐观赏Web Operations 101: Web Flow
  • 对RDBMS关联式数据库和SQL语法有基本了解，推荐阅读MySQL 超新手入门从第0章至第5章CRUD与资料维护。
  • 熟悉指令列操作(CLI: command line interface)，Unix-like作业系统为佳，推荐The designer’s guide to the OSX command prompt和The Command Line Crash Course教材。

如果您有PHP/MySQL或ASP、JSP等网页应用程式开发经验，相信这些对您应该不陌生。
