Git 版本控制系统

Git 是 Ruby/Rails 世界中最为通行的版本控制系统，包括 Rails 本身以及绝大部分的套件都是使用 Git 来做版本控制，甚至可以说大部分都使用 GitHub 这个服务，GitHub 是基于 Git 这套分散式版本控制系统的 Repository hosting 应用，只要是开放原始码软件，都可以免费的使用这个服务。

更多信息请前往笔者的版本控制系统 Git一书。
